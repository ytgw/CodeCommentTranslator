Source Code Comment Translator

使い方

  1. プログラミング言語を選択する。 Customを選択した場合は、言語設定を入力をした後に「入力内容を設定」ボタンを押す。
  2. ソースコード入力フォームに、翻訳したいソースコードを入力する。 例えば このJavaScriptコード を入力する。
  3. 「前処理」ボタンを押し、ソースコードからコメントを抽出し、前処理結果フォームに出力する。
  4. 必要に応じて前処理結果フォームの内容を修正する。
  5. 「翻訳サイトへ」ボタンを押し、外部サイトに前処理結果を送り、翻訳する。
